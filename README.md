# chatbot-project
# 🤖 NLP Chatbot using Hugging Face Transformers

## 📌 Project Overview
This project is a simple chatbot built using Hugging Face Transformers.  
The chatbot interacts with users and generates responses using a pre-trained model.

---

## 🧠 Technologies Used
- Python
- Hugging Face Transformers
- PyTorch
- Jupyter Notebook

---

## ⚙️ How it Works
- The chatbot takes user input from the console
- Converts text into tokens
- Uses a transformer model to generate a response
- Displays the response
- Continues conversation until user types "exit"

---

## 💡 Features
- Interactive chatbot
- Uses pre-trained transformer model (DialoGPT)
- Continuous conversation
- Exit option included

---

## ▶️ How to Run
1. Install required libraries:
   pip install transformers torch

2. Run the notebook

3. Start chatting:
   Type your message and get response

4. To stop:
   Type "exit" or "quit"

---
